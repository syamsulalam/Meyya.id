# MEYYA Subagents

This folder contains reusable job descriptions for bounded Codex subagents in this repo.

Use these files as prompt/context references when delegating work:

- `frontend-ui-worker.md`: focused React, Tailwind, tooltip, layout, icon, and accessibility fixes.
- `docs-roadmap-worker.md`: documentation, roadmap, changelog-style updates, and copy audits.
- `api-schema-reviewer.md`: backend API, D1 query, schema, validation, and edge-case review.
- `qa-verifier.md`: lint/build/test verification and focused regression checklists.

The primary operator remains responsible for final integration, production-risk decisions, and the final answer to the user.
